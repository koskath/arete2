# Slide 31 of Lecture 14 contains information about the XAI.

• Explainability aims to answer stakeholder questions about the decision-making
processes of AI systems.
• Use explanations to ensure project requirements are met during building,
debugging, and testing.
• Transparency is important thanks to current context of rising AI ethical concerns.
• XAI is a set of processes and methods that allows humans to comprehend and trust
results created by ML algorithms.
• Helps promote user trust, model auditability and productive use of AI.
• Mitigates compliance, legal, security and reputational risks of production AI.
• Key requirements: fairness, model explainability and accountability.
